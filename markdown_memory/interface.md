0. 测量数据显示页面：
    - 测量数据显示页面即为有绿色弧线的图一
    - 最上方一行显示SQ，显示格式是“SQ: X.XX”，也就是前面是最多整数，后面是固定的两位小数
    - 中间通过ARC模块显示流量，旋钮消失掉，背景弧为#24FC3D颜色，前景弧为白色
    - 在ARC模块中间显示当前流量大小，ARC模块的角度是通过流量大小变换前景弧占背景弧的占比
    - 最下方一行显示累计流量，单位是“m3”，前面用6位小数显示数据，也就是整数位根据需要扩展，小数位固定为六位

1. 主菜单页面：
    - 见图2
    - 在测量数据显示页面按下BACK按键之后切换到主菜单界面，背景为黑色，其最上面留出来一行左侧 显示“菜单项”文字，右侧在进入时显示<1/X>，这里的X是菜单项的数量除以3的结果
    - 主菜单界面会显示多个菜单项，菜单项的样式是灰黑色的圆角矩阵，其中间是文字，显示菜单项内容
    - 进入菜单页面之后，默认选择上数下第一个菜单项，被选择的菜单项的圆角矩阵周围会出现亮绿色border以示选中
    - 主菜单有多个菜单项，但是在页面中每次最多只会显示三个菜单项，主菜单页面可以通过按键切换，其具体切换形式是：
    - 进入菜单项，默认选择第一项，通过按键UP和DOWN切换当前选择的菜单项，在第一项时，按下UP按键无反应，按下DOWN按键的时候向下切换选择的菜单项，当当前选择上数下第三个菜单项的时候，继续按下DOWN按键，右上角的<1/2>会切换成<2/X>，然后显示后三个菜单项，如果数量不足三个，就显示剩下的菜单项数量

2. 子菜单项：
    - 在选择主菜单页面上的菜单项之后，按下OK按键，可以进入该菜单项对应的界面，可以是子菜单项，或者是直接进行数据设置（进入数据设置界面）。具体的数据设置界面可以在另外的文件中另写，所以要保留一定的框架。如果是子菜单项，那么其界面类似主菜单页面的样式，左上角显示当前子菜单项名字，右上角根据该子菜单内项目数显示<1/X>，示例见图三

- 若当前在主菜单界面，则不论当前在哪个菜单项，若按下BACK键则返回到测量数据显示页面。若当前在子菜单界面或者数据显示界面，则返回上一界面，选择项默认为上一菜单的第一项。
